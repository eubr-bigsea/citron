Caso de Uso

A operação Escritor de Dados é utilizada, geralmente, para escrever os dados que foram transformados na ferramenta em uma nova fonte de dados, que poderá ser usada no futuro sem a necessidade de execução das transformações novamente, ou baixada pelo usuário na aba Datasources.

Vamos supor que precisássemos salvar uma base de dados que seja uma versão modificada de uma base já existente. Para este exemplo utilizaremos a base Titanic, e selecionaremos os atributos que queremos salvar na nossa nova base de dados.

1) Primeiramente, vamos utilizar a operação Leitor de Dados para ler a base de dados Titanic, marcando a segunda caixa da aba de Resultados, a fim de comparar com a base que será criada.

 2)  Utilizando a operação Projeção/Selecionar atributos, vamos selecionar os atributos class, survived, name, sex e age, que denotam, respectivamente, a classe do passageiro em questão, sua sobrevivência ou não ao desastre, seu nome, seu gênero e sua idade.
